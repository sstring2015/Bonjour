## General info
This is a video chat web app,where user can make a call to his/her friend.
	
## Technologies
Project is created with:
* WebRTC
* Socket.io
* Node.js
* React.js
	
## Setup
To run this project, install it locally using npm:

```
$ cd Bonjour
$ npm install
$ cd client
$ npm install
```
Then type
```
$ npm start
```
in server as well as client side to run the project
